gypsum
======

Lorem Ipsum generator for your Go app.

## Usage:

Install the package.

``` bash
go get github.com/aybabtme/gypsum
```

In your go code, import the package:
``` Go
import "github.com/aybabtme/gypsum"
```

Then use it:
``` Go
aSinglePara := gypsum.Lorem()
tenPara := gypsum.ManyLorem(10)
```
